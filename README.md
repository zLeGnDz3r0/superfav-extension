# superfav-extension
💎 SuperFav for Twitch — extensión gratis para Chrome. Marca streamers con un clic, badge 🔴 de directos, popup con toggles y avisos de escritorio 🖥️ (directo + cambio de título). Privado: tus favoritos solo en tu navegador. Compatible con Edge y Brave. 🏪 Próximamente en Chrome Web Store.
